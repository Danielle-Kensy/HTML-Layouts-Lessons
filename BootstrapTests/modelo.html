Bootstrap - Framework Front-end

O bootstrap é um framework profissional voltado para público de desenvolvedor front-end. O mesmo é dotado de diversos recursos de estilização CSS, efeitos JS, além de jQuery. Para utilizar o bootstrap é necessário vincular ao arquivo bootstrap.min.css e bootstrap.min.js ou utilizar por CDN - links.

Exemplo de Botão:

<button type="button"> Botão </button>

Exemplo de Botão Bootstrap:

<button type="button" class="btn btn-default"> Botão </button>
